DEFINITION

NDTMP specifies the nodal temperature to be applied to individual nodes.

 

REMARKS

If no value is specified for DefTemp, it is assumed to be zero.

 

Nodal temperature values may not be applied to nodes which have heat transfer boundary constraint code 1 (BCCTMP).

 

Nodal temperature may be specified as a function of time with keyword BCCFNC.

 

The NumColumns variable allows more than one temperature to be specified for a given node.  This is used in the case of storing temperature gradient information between steps.

 

Applicable object types: : Rigid, Elastic, Plastic, Elastoplastic, and Porous.
