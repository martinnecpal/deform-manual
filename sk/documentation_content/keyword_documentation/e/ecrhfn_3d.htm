DEFINITION

ECRHFN specifies whether a resistance heating face boundary condition function is defined as a function or through a user subroutine.

 

REMARKS

The function type (Ftype) options are:

< 0 user routine number

> 0 function number (specified in BCCFNC)

The user should be aware that the face definition function shares the function structure with the nodal boundary condition function keyword (BCCFNC).

 

Applicable object types: Elastic, Plastic, Elastoplastic, Porous
