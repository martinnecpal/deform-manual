GenerateMethod

Geometry generation method

0 - Generate All

-1 - Do not generate

1 - Generate 2D geometry or 2D cross section only

2 - Convert 2D cross section to 3D geometry

3 - generate 2D cross section and convert to 3D geometry

4 - create 3D geometry directly

None

 

DEFINITION

GENGEO is an action keyword that generates geometry from parameters defined in a GEOPR2 or GEOPR2 keyword.

 

REMARKS

When GenerateMethod is 2 or 3, GENGEO will also create the WPAXIS (type 4) which defines the extrusion or revolve that creates the 3D geometry from it's 2D cross section.

 

RELATED TOPICS
