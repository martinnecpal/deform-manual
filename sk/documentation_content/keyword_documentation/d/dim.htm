Update History:

Last updated on : 23-07-2013

 

DIM  Dimension

  

OPERAND

DESCRIPTION

DEFAULT

Dimension

Dimension of model

None

 

DEFINITION

DIM specifies the dimension of the importing keyword file.

 

REMARKS

This is special action keyword which sets dimension of the imported keyword. It is introduced to handle dual database step data in 2D/3D integrated pre-processor.
