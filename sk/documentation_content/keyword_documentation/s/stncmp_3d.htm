DEFINITION

STNCMP specifies the value of strain components at the centroid of each element.

 

REMARKS

The field width of strain components is determined by strain component storage flags specified by STNOUT.

 

The possible components include Plastic, Elastic, Creep, Transformation  plasticity, Thermal, Transformation volumetric, and Total.

Elemental strain components are interpolated between meshes during remeshing procedures.

 

Applicable object types: Plastic, Elastoplastic, Porous
