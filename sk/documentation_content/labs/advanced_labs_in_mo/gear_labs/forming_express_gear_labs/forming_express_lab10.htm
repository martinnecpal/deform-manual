Lab 10. Die stress

Open Project

 Open the Project FE_Gear_LargerBillet.

Add Die stress study

In this lab, we will preform Die stress operation for the dies using Die stress study operation, so click on  button and select Die stress study from the list (See Fig. L10.1.).

 

 

Adding Die stress study from list

 

Step selection

Select the last step in step list for die stress analysis (See Fig. L10.2.), then Click  until top die general page.

 

 

Step selection window

Define Top die data

Verify the Top die temperature is 300°F. By default, the object type of rigid objects from the loaded step will be changed to elastic. Keep the object type as elastic for the Top Die object. From v13.1.1, two new options have been introduced in the object page to position the dies in future steps for Multiple steps die stress analysis, we need not use these options for single step die stress analysis, hence, keep the Need positioning check box turned off. Click  until mesh page.

 

 

Top Die page

Generate Top die mesh

Set the target number of elements to 1000 and generate mesh. Click  until Material page.

Assign Top die Material

In Material window, assign AISI-H-13 material from material library's, from Die_material catagory and click   to BCC page.

Assign BCC for Top die

In BCC page, check the default assigned Deformation BCC  for Left side of the Top die in X-Direction, then Apply Vy = 0 Velocity boundary condition on the Top surface of the Top Die (See Fig. L10.4.). Click  to Force interpolation page.

 

 

Top die BCC window

Interpolate Force for top die   

In Force interpolation page, click on  action label to interpolate forces as shown in Fig. L10.5. Click  to Bottom Die page.

 

 

Top die Force interpolation window

Bottom die data

Verify the Bottom die temperature is 300°F. We will keep the object type as Elastic for the Bottom Die and turn off Need positioning check box. Click  until mesh page.

 

 

Bottom Die page

 

Generate Bottom die mesh

Set the target number of elements to 1000 and generate mesh. Click  until Material page.

Assign Bottom die Material

In Material window, AISI-H-13 material from material library and click  to BCC page.

Assign BCC for Bottom die

In BCC page, check the default assigned Deformation BCC for Left side of the Bottom die in X-Direction, then Apply Vy = 0 Velocity boundary condition on the Bottom surface of the Bottom Die (See Fig. L10.7.). Click  to Generate DB page.

 

 

Bottom die BCC page

Interpolate Force for Bottom die   

In Force interpolation page, click on  action label to interpolate forces as shown in Fig. L10.8. Click  until Generate DB page.

 

 

Bottom die Force interpolation window

Generate DB

Click on  action label to check the problem. Generate a database by clicking   action label.

Save the project and click on the MO  Mode button (above the operation tree) to start the simulation.

Run and monitor a simulation

Click on the  action label to open the Run Options dialog. Use the default Continue Run option to select “Continue from the last step” option and then select the Simulation mode as Interactive radio button. Click on button to run the simulation.

Review the results

When the simulation is completed, click the MO  Mode button to review results.

 

In the Post-processor, click on the positive step in the step list.

Plot Effective stress state variable (See Fig. L10.9.).

 

 

Effective stress State variable Plot

 

Go to the Set up  dialog, and display Max principal stress (See Fig. L10.10.) and other stress components.

 

 

Max Principal Stress state variable plot

 

Switch to  mode and add another Die stress study to add Shrink Fit. Click Lab 11. Die stres with a press fit. to continue Shrink fit lab.
