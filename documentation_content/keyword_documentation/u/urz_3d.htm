DEFINITION

URZ specifies the nodal speed of each node.

 

REMARKS


The nodal speed is defined in the local coordinate system specified in BCCANG.

 

Typically, the speed constraint is used for limiting translational degrees of freedom along an axis of symmetry. This is achieved by specifying a speed of 0.0 in the direction perpendicular to the axis of symmetry.

 

Most process related object motion, such as die speed, can be specified with movement control constraints (MVOCTL). However, speed constraints are occasionally used for operations which require an object to be pulled through a set of dies, as can be the case with drawing and extrusion processes.

 

Applicable object types: Elastic, Plastic, Elastoplastic, and Porous.
