# 6.6.4. Upgrading Operations

Forming express operation have basic options to setup the operation easily with less time, if the user needs to use advanced options while setting up or after simulation of the forming express operation then forming express operation can be promoted to forming operation to access advanced options. In the operation editor, if user right clicks on the forming express operation then it will provide the option to promote to forming operation as shown in Fig. 6.6.4.1.

![]({{ '/assets/images/integrated_manufacturing_process_setup/6_6_operations_management/6_6_4_upgrading_operations/6_6_4_image001.jpg' | relative_url }})

Forming express operation with promote to forming operation option

User will be asked to confirm the promotion as shown in Fig. 6.6.4.2. Confirming the promotion will promote to forming and user can observe the promoting process status in status bar below the property editor as shown in Fig. 6.6.4.3.

![]({{ '/assets/images/integrated_manufacturing_process_setup/6_6_operations_management/6_6_4_upgrading_operations/6_6_4_image002.jpg' | relative_url }})

Confirm promotion popup 

![]({{ '/assets/images/integrated_manufacturing_process_setup/6_6_operations_management/6_6_4_upgrading_operations/6_6_4_image003.jpg' | relative_url }})

Promotion status bar

After promotion user can access the advanced options available in forming operation as shown in Fig. 6.6.4.4.

![]({{ '/assets/images/integrated_manufacturing_process_setup/6_6_operations_management/6_6_4_upgrading_operations/6_6_4_image004.jpg' | relative_url }})

Forming express operation after promoting it to forming operation

The advanced options which will available in after promoting to forming operations are geometry types like plane stress and torsion in 2D, object types like Elasto plastic, elastic and porous, advanced geometry, advanced mesh, object properties and advanced simulation controls options including the different stopping controls, solvers selection, remesh options etc.

For more details on options available in forming operation refer the chapter[ Introduction to Forming Operation.](/docs/en/operation_templates/33_forming/33_introduction_to_forming/)
