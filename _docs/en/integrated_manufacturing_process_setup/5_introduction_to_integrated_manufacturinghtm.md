# 5\. Introduction to Integrated Manufacturing process (MO)

Integrated Manufacturing process (MO) is an integrated form of all the current templates. It is a foundation for parametric/DOE studies and optimization. From Cogging to Lifing different manufacturing process like Forging, Heat treatment, Machining, Distortion and Lifing can be unified easily using Integrated Manufacturing process (MO) environment (See Fig 5.1) without any data transfer problems between operations.

![]({{ '/assets/images/integrated_manufacturing_process_setup/5_introduction_to_integrated_manufacturing/5_image001.jpg' | relative_url }})

MO Environment

The existing modules have been made available in MO environment as,

  * Cogging
  * Die-Stress
  * Extrusion
  * Forming (General purpose forming)
  * Forming Express (F2/F3)
  * Heating Express
  * Heat Treatment
  * Furnace heating
  * Machining
  * Machining distortion
  * MultiBlow
  * Ring rolling
  * Shape rolling
  * Spinning
  * Report Generation
  * Convert 2D to 3D
  * Cyclic Operation
  * Design of Experiments (DOE)
  * Optimization,
  * DEFORM-Post (Next Gen. Post)
  * Material Suite
  * Data Analytics
  * Inverse heat
  * 3D Geometry tool are in the process of converting to MO modules.
