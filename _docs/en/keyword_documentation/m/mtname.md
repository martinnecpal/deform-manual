# MTNAME

|  (Material Data)  
---|---  
|  Last updated on : 08-08-2013  
  
* * *

MTNAME Material

MatName

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Material |  Material number |  None  
Material name |  Material |  Material   
Note: Material character string up to 64 characters.  
  
DEFINITION  
---  
MTNAME assigns a material name to a particular material number.  
  
REMARKS  
---  
Applicable simulation types: Isothermal Deformation Non-Isothermal Deformation
