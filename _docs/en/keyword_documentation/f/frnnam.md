# FRNNAM

|  (Simulation control)  
---|---  
_Update History:_ (New keyword in v11) |  Last updated on : 23-07-2013  
  
* * *

FRNNAM FurnaceName

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
FurnaceName |  Object Number |  None  
  
DEFINITION  
---  
FRNNAM assigns the name of particular furnace equipment.  
  
REMARKS  
---  
Heating in the furnace can be modeled using environment temperature (ENVTMP) in Simulation controls dialog.
