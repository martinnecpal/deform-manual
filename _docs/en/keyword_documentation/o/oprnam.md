_Update History:_ V11 - OPRNAM has been introduced. |  Last updated on : 24-07-2013

OPRNAM

Operation_name

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Operation_name |  Name of operation |  None  
  
DEFINITION  
---  
OPRNAM specifies the operation name.  
  
REMARKS  
---  
An operation name can be given to each specific process (or operation). It is typically used for multiple operations.
