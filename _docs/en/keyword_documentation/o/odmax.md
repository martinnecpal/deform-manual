# ODMAX

|  (Simulation control 3D)  
---|---  
|  Last updated on : 13-08-2013  
  
* * *

OBJTYP Object, Otype, Ftype

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
OuterDiameterOfRing |  stopping criterion of maximum diameter(radius) of ring |  0.0  
  
DEFINITION  
---  
ODMAX limits the outside diameter of the ring object in the entire simulation. The simulation will be stopped when the outside diameter of the ring object reaches OuterDiameterOfRing,  
  
REMARKS  
---  
Applicable [simulation types](/docs/en/pre_processor/9_simulation_controls/9_1_simulation_type_settings/): Ring Rolling Deformation
