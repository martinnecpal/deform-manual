# GENCTC

|  (Action keyword)  
---|---  
|  Last updated on : 07-08-2013  
  
* * *

GENCTC Tolerance

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Tolerance |  Contact tolerance value |  0.0001  
  
DEFINITION  
---  
GENCTC generates the contact conditions for the specified object.  
  
REMARKS  
---  
GENCTC is an action keyword. The contact conditions are generated prior to database generation.
