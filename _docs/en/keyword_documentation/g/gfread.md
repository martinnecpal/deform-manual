# GFREAD

|  (Action keyword)  
---|---  
_Update History:_ V11 - GFREAD has been introduced. |  Last updated on : 25-07-2013  
  
* * *

GFREAD Type(=1)

FileName

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Type |  File type |  1  
FileName |  File name to read |  DEF_MORP_OUTPUT.GEO  
  
DEFINITION  
---  
GFREAD reads the target geometry file created from morphing in 3D shape optimization.  
  
REMARKS  
---  
This is action keyword that reads the target geometry which is generated from source geometry using morphing technique during 3D shape optimization run.
