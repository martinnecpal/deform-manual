# CNTRAX

|  (Object data – 3D)  
---|---  
|  Last updated on : 12-08-2013  
  
* * *

CNTRAX Obj, arg1..arg7

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Object |  Object number |  None  
Arg 1-3 |  Rotational axis point |  None  
Arg 4-6 |  Rotational axis vector |  None  
Arg 7 |  Current angle |  None  
  
DEFINITION  
---  
CNTRAX specifies the center of rotation for objects which move with a rotational velocity.

---
