CRDSYS

(Action keyword)

Update History:

(New) Definition has been introduced in v10.0

Last updated on : 23-07-2013

 

CRDSYS  Object, AxisUp

  

OPERAND

DESCRIPTION

DEFAULT

Object

Object number

None

AxisUp

Coordinate system definition

=3 Z is up: X(R)->X, Y(Z)->Z

=2 Y is up: X(R)->X, Y(Z)->Y

None

 

DEFINITION
CRDSYS specifies the 3D coordinate system setup when 2D model is converted to 3D model.

 

REMARKS

This is action keyword usually written in DEF_MULTI.INI which is input file to M23.EXE binary which converts 2D model to 3D.

 

RELATED TOPICS

2D to 3D model conversion, M23.EXE
