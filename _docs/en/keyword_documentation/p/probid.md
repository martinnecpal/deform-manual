# PROBID

|  (Action keyword)  
---|---  
|  Last updated on : 13-08-2013  
  
* * *

PROBID

Problem-id

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Problem-id |  Sets the Problem ID during the processing of keywords |  None  
  
DEFINITION  
---  
The PROBID variable Sets the Problem ID during the processing of keywords.  
  
REMARKS  
---  
It is an action keyword placed in either an automatic script file or in a Master file. The purpose of PROBID is to set the problem id during the processing of a master file in order to specify the database to be read and/or written to.
