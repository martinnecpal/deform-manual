# PRSNAM

|  (Object Data)  
---|---  
|  Last updated on : 09-08-2013  
  
* * *

PRSNAM Object

Name

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Object |  Object Number |  None  
Name |  Press Name character string up to 64 characters |  None  
  
DEFINITION  
---  
PRSNAM provides a description of the movement control. Usually it is associated with a press.
