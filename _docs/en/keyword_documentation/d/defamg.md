# DEFAMG

|  (Action keyword)  
---|---  
_Update History:_ V11 - DEFAMG has been introduced. V11.1 - Added option to not generate default BCC |  Last updated on : 29-07-2013  
  
* * *

REMESH Object

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Object |  Object number |  None  
NoDefaultBCC |  Controls generation of Default BCC 0 - Generate default BCC 1- Do not generate default BCC |  0  
  
DEFINITION  
---  
REMESH specifies the object number for mesh generation By deafult DEFAMG generates default BCC. Setting the NoDefaultBCC parameter to 1 prevents this.  
  
REMARKS  
---  
This is action keyword that executes mesh generation.
