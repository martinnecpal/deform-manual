DEFINITION

DYNARY allocates dynamic arrays in which FEM can save built-in flownet and Lagrangian mesh definition for ALE simulation.

 

REMARKS

The data defined by use of this keyword is not saved in A array or MAR array, but instead it is saved in dynamic array region in the database file. This keyword allocates the space for state variables for nodes as well.
