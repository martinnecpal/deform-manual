DEFINITION

SOLMTD specifies the solver type used in deformation simulation.

 

REMARKS

The skyline solver uses the skyline storage method in conjunction with Gaussian elimination.

The sparse solver is optimized for sparse matrix problems.

 

MUMPS solver is available from DEFORM-3D v11. It is recommended for induction heating simulations in which stiffness matrix is un-symmetric and has a complex value.

 

Applicable simulation types:  Isothermal/Non-Isothermal Deformations
