# 53\. Introduction to Optimization

Optimization (OPT) approach is an to achieve robust process design optimization. Using this wizard user can perform optimization of the process and preform workpiece and die shapes. This is done by inputting the variables drive uncertainties and its limits (DOE helps to identify the limits of such variables for optimization) and setting the objective function and constraints (variable and fill constraints). The optimization wizard gives the optimized process and/or shape as output.

MO Optimization is coming more user friendly interface with the new features those are,

  * Next generation optimization controller has been developed

  * More objective functions are added

  * Multiple region-of-interests can be handled

  * Now multiple operations can be optimized due to MO environment

  * Intuitive way of defining optimization variables due to DOE environment

  * Optimizing both process condition and shape together

  * Unified user interface for both 2D and 3D optimization

  * Specially designed “Optimization Post-processor” has been developed
