45. Introduction to Boolean Operator

Boolean operator is used to remove the unwanted area of the object or material like flash or piercing operation. Using this user can even increase the computational efficiency of subsequent operations simulation. This operator available in simulation operators group in MO explorer. User can setup this operation after or before the previous operation simulation, but as of now only batch mode support this operation. In this manual 2D Boolean operation explained in without simulating previous extrusion operation and 3D Boolean operation explained once after simulating the previous operation.
