Lab 5. 2D to 3D conversion

Creating a New Problem

Create a new project called Gear23 and copy the existing project Gear_LargerBillet.

 

 

MO New Project window

Adding operations

Right-click on the 2nd operation tile in operation editor and delete it. Add the following operations:

  • [2D] Heat Transfer Express

  • 2D to 3D Converter (located in Simulation Operator)

  • [3D] Forming Express

 

 

Operations list in Operation Editor

Upset Operation

In the initial Upset tile in Operation Editor, right-click on the Workpiece and Pass object to All operations.

Select Upset operation and Click . Go to Simulation controls and check Heat transfer mode.

 

Generating DB
Select Generate DB from the operation tree and Generate the Database. Click  to Heat Transfer Express operation.

Heat Transfer Express operation

Process

In Process page, select the heating type as Transfer. Click  to Object list.

Object selection

Select 1 workpiece only radio button. Click  to Heat condition.

Heat Condition

Set the transfer time to 5 seconds and accept the default temperature and convection coefficient. Click  to Simulation controls.

Simulation Controls

Use the default value and click  until 2D to 3D Convertor operation.

Convert 2D to 3D operation

Verify Z is up (consistent with other models) and  in 3D parameters start angle and Revolve angle should be 0 and 360 to generate whole workpiece. Click  to Workpiece geometry page.

Workpiece Geometry

The # of sections sets the number of geometry sections around the circumference. One section per degree of rotation works well, so set the # of sections to 360. click  to workpiece mesh page.

Workpiece Mesh

Leave the Mesh at the default 32000 elements. click  to workpiece Material page.

Workpiece Material

Leave the Material as (inherited) in Material page. click  until Forming express Top die general page.

3D Forming Express

Top die general

Keep the default 300°F temperature for Top die, click  to Top die geometry page.

Top die geometry

Click the  (Load geometry from a file) button and  open (import) the file “gear top die.STL”. The geometry for top die is located in DEFORM installation folder  \Tutorials directory. Define a reference point on the flash land using  option. click  to Top die movement page.

Top die Movement

Set the Speed to a constant value of 3 in/sec in the -Z direction. click  to Bottom die page.

Bottom Die general

Keep the default 300°F temperature for bottom die, click  to bottom die geometry page.

Bottom die geometry

Click the  (Load geometry from a file) button and  open (import) the file “gear bottom die.STL”. The geometry for gear  bottom die is located in DEFORM installation folder  \Tutorials directory. Define a reference point on the flash gutter using  option. click  to positioning page.

Positioning dies

Rotate both dies 90° about the X axis so that they line up with the Workpiece. Click  to Scheduled positioning page.

Scheduled positioning

Click on  in scheduled positioning page and you can see the positioning scheduled as shown in Fig. L5.3.  Click  until stopping controls page.

 

 

Scheduled positioning window

Stopping controls

Set the Distance between objects to 0.25 in. and set the Top Die and Bottom Die as reference objects. Click  to assign Primary die stroke page.

Primary die stroke

Define Primary die stroke value as 1.5 in input field. Click  to Simulation controls page.

Simulation controls

Set Number of steps to 1000, Step increment to 5, select User type step definition method and define constant die displacement as 0.02 in.

 

Save the project and click on the MO  Mode button (above the operation tree) to start the simulation.
