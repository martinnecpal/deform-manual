# 26.6.10. **CA model setup**

Cellular Automata (CA) models are synchronous algorithms that describe the discrete spatial and temporal evolution of complex systems by applying local (or mesoscopic, mid-range) deterministic or probabilistic transformation rules to lattice cells with local connectivity. For more information please refer [Material Suite Labs for Cellular Automata (CA) model](/docs/en/labs/material_suite_labs/material_suite_labs_for_ca_model/) .
