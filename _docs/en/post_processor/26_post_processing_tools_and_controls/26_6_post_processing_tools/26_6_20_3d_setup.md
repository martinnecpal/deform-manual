# 26.6.20.3D Viewer and 3D Setup parameter

  * 3D Viewer

  * 3D Setup Parameter

  
**3D Viewer [2D]![]({{ '/assets/icons/post_icons/mo_3d_view_icon.jpg' | relative_url }})**

The 3D viewer option is used to view the 2D simulation results by revolving or extruding the objects into 3D. (See Fig. 26.6.20.1.)

For the axi-symmetric or torsion 2D geometry type as shown in Fig. 26.6.20.1. 3D view is generated by revolving the 2D objects using 3D view setup parameters and these parameters can be varied as shown in Fig. 26.6.20.3. See Fig. 26.6.20.2. that displays extruded 3D view of the 2D plane strain or plane stress objects.

**3D Setup Parameter****[2D]** ![]({{ '/assets/icons/post_icons/mo_3d_setup_icon.jpg' | relative_url }})  
The users can view results for the 2D simulation in 3D either by sweeping the geometry to the required angle for 2D axi-symmetric or torsion geometry type or Extrude the geometry in the required direction for 2D plane strain or plane stress geometry type. The mesh sections can be increased or decreased by setting the resolution to high, medium or low, different sweep angles can be selected and start angle can be varied in case of axi-symmetric or torsion geometry type as shown in the Fig. 26.6.20.3.For plane strain or plane stress geometry type length of extrusion and extrusion start location can be varied as shown in Fig. 26.6.20.4. The upward axis in 3D view can be either Y or Z as shown in Fig. 26.6.20.4. These modifications can be applied to multiple objects or selected object.
