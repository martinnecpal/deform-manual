# 1.1. Overview of DEFORM

DEFORM have different wizards for specific processes like [Shape rolling](/docs/en/operation_templates/43_shape_rolling/43_introduction_to_shape_rolling/), [Ring rolling](/docs/en/operation_templates/42_ring_rolling/42_introduction_to_ring_rolling/), [Extrusion](/docs/en/operation_templates/31_extrusion/31_introduction_to_extrusion/), [Spinning](/docs/en/operation_templates/48_spinning/48_introduction_to_spinning/), [Inverse HT,](/docs/en/inverse_heat/51_introduction_to_inverse_heat/) [Machining](/docs/en/operation_templates/39_cutting/39_introduction_to_cutting/), [Cogging](/docs/en/operation_templates/29_cogging/29_introduction_to_cogging/), [Heat treatment](/docs/en/operation_templates/37_heat_treatment/37_introduction_to_heat_treatment/), [HT Furnace](/docs/en/operation_templates/38_furnace_heating/38_introduction_to_3d_ht_furnace/), [Forming Express](/docs/en/operation_templates/34_forming_express/34_introduction_to_forming_express/) and [Forming](/docs/en/operation_templates/33_forming/33_introduction_to_forming/), these wizards are customized and helps user to setup complex processes with ease.

[Integrated Manufacturing Process](/docs/en/integrated_manufacturing_process_setup/5_introduction_to_integrated_manufacturinghtm/) interface helps user to construct successive operations initially and simulate all operations in sequence without user interaction.

[DEFORM-HT](/docs/en/operation_templates/37_heat_treatment/37_introduction_to_heat_treatment/) adds the capability of modelling heat treatment processes, including normalizing, annealing, quenching, tempering, aging and carburizing. [DEFORM-HT](/docs/en/operation_templates/37_heat_treatment/37_introduction_to_heat_treatment/) can predict hardness, residual stresses, quench deformation, and other mechanical and material characteristics important to those that heat treat.

[DOE](/docs/en/doe_and_optimization/52_doe_study/51_introduction_to_doe/) helps the user to study effect of various parameters on process in a specified range. [OPTIMIZATION](/docs/en/doe_and_optimization/53_optimization_study/53_introduction_to_optimization/) helps the user to optimize specific parameter like Die Load, Max. Strain in billet, Damage value etc.

Ultimately, DEFORM is able to study the entire manufacturing chain from ingot conversion, through forming, machining and heat treatment, to final product installation. At the same time, the modern user interface has been designed to be accessible to both production engineers and research scientists.
