Lab 11.Add a case with a press fit

Step selection

Select the last step in step list, then Click  to Objects page.

Add object

In the Objects table, add a 3rd object. Click  to New object page.

Top die and Bottom die

Repeat the same mesh/interpolation/material/BCC procedure as explained in Lab 10: Die stress for the Top Die and Bottom Die. After completing the setup,  click   to Object 4 page.

Object 4 General page

Change the object name to Case and set the temperature to 300°F. We will select the object type as Elastic for the Case and turn off Need positioning check box. Click  to geometry page.

 

 

Case object page

 

Create Case Geometry

Use Define primitive to create a hollow cylinder for case with inside radius 8.5 in, Outer radius 14.4 in, and height of 5.4 in.( See Fig. L11.2.)

 

 

Case geometry primitive window

Generate Case Mesh

Generate a mesh with 250 elements. Since the workpiece does not contact this case, no force interpolation is required. So, click  until Material page.

Assign Material for Case

In Material window, assign AISI-H-13 material from material library's, from Die_material catagory and Click   to BCC page.

Assign BCC

We will assign a shrink fit boundary condition along the ID. Click Shrink Fit in the boundary condition list, and then use the By edge   tool to select the ID of the Case. Next, select User type and input a value of 0.002 in (See Fig. L11.3.). for both Interference input fields. Choose the x direction as the shrink fit direction. Click .

 

 

Selecting the ID to define Shrink fit BCC

 

Note: The purpose of the two input fields it to allow variable interference along the defined boundary.  The arrow near the center line displays the start and end point of the selected boundary. Since we are doing a uniform interference, we chose the same value for both start and end points.

 

A pop-up window will ask if you want to move nodes by the offset distance.  

If tools are drawn true to size (i.e. overlapping), then we need to move the nodes. In this case, they are already drawn as matching surfaces, so DO NOT MOVE the nodes (i.e. answer No) (See Fig. L11.4.). Assigned Shrink fit BCC for Case object is as shown in Fig. L11.5.

 

 

Move object Nodal coordinate popup

 

 

Assigned Shrink fit BCC for Case

 

Positioning Case

Go to Positioning and Drag the Case down until the bottom surface of the Case and Bottom Die are aligned. Click  to Contact page.

Contact

Add one relation by clicking on  button, select Maser as Case object and Slave as Bottom Die object. Define friction value as 0.1. Click on  to generate contact.(See Fig. L11.6.) Click  to Simulation controls page.

 

 

Inter object relation windows

Simulation Controls

Default simulation controls are acceptable for this model. Click  to generate DB.

Generate DB

Click on   action label to check the problem. Generate a database by clicking  action label.

 

Save the project and click on the MO  Mode button (above the operation tree) to start the simulation.

Run and monitor a simulation

Click on the  action label to open the Run Options dialog. Use the default Continue Run option to select “Continue from the last step” option and then select the Simulation mode as Interactive radio button. Click on button to run the simulation.

Review the results

When the simulation is completed, click the MO  Mode button to review results.
