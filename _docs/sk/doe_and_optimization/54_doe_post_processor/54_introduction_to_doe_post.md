# 54\. Introduction to DOE Post

DOE Post-processor provides options to interpret the output of variables selected in DOE Output / Shape Optimization while setting up DOE/ Optimization study. During the simulation of DOE/ Optimization Study several DBs are generated based on the DOE Sampling or objective function. DOE Post-processor provides graphical options to interpret results from all these DBs in a Single display area. At the end of DOE/ Optimization study DEFORM generates an .SDB file by extracting the respective data based on the different outputs selected in the DOE Output while setting up the problem. This .SDB file is available in Output folder of the respective DOE/ Optimization study. The DOE Post-Processor looks like as shown in Fig. 54.1.
