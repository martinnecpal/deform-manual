Creating a New problem

On a Windows machine, go to the  button select DEFORM-v1x.xxx (.xxx indicates version number E.g. v14.0.2) and select DEFORM GUI Main v1x.xx. from the menu. The DEFORM GUI Main window will appear.

Create a new problem either by selecting File  New Problem or by clicking the New Problem  icon. The Problem Setup window will appear as shown in Fig. 3DLFL.1. Select " Integrated Manufacturing Process" radio button and units system as "SI" radio button in units field. Define Problem Name as "3D_Laminar_Flow" and make sure the “Show option dialog” check box is turned on (if we do not turn on the “Show option dialog” check box, then we will not get the New Project dialog in MO UI). Then click on   button to open a new Problem using the Deform Integrated Manufacturing Process.

 

 

Problem type selection window

 

Multiple operation wizard will open with the New Project dialog, at this point user will be prompted to specify a project name (system will create a separate folder with this project name) and title for this session. In this session we use ‘3D_Laminar_Flow’ as the project name. Click on  to continue to open the operation.

Adding 3D Forming operation

Add one 3D Forming operation from the Explorer Operations list. Operation can be added by clicking on 3D Forming operation  button or user can also add by drag and drop into the Operation Editor (See Fig. 3DLFL.2.).

 

 

Adding 3D Forming Operation

Simulation controls

Turn ON ‘Heat transfer’ and ‘CFD flow’ options and uncheck ‘Deformation’ as shown in Fig. 3DLFL.3., then click on .

Add new material ‘Ideal fluid’

On ‘Material list’ page, click on  to add a new material. Double click on the new material name tag, change the material’s name to ‘Ideal fluid’ (See Fig. 3DLFL.4.). Click to material page to edit the material parameters.

 

 

Material List page

 

Click on ‘Elastic’ tab, then type the Young's modulus value as 1 as shown in Fig. 3DLFL.5. for elastic material data. 

 

 

Material page - Elastic data


 Click on ‘Thermal’ tab, then type the values as shown in Fig. 3DLFL.6. for thermal/flow material data. Click on  to Object page.

 

 

Material page - Thermal data

Define object

This lab only needs one object, so on ‘Object’ list page, removes all objects except ‘Workpiece’, then click on . (See Fig. 3DLFL.7.)

Import Object

In ‘Workpiece’ page, click on  ‘Import Object from a file’ then choose ‘Flow_cylinder_3D.KEY’ from DEFORM installation folder \3D\LABS\ directory folder an select object "Workpiece" in object selection page to import. (See Fig. 3DLFL.8.)

 

   

 

Import object

 

Choose ‘Environment - Fluid(CFD)’ as object type with Object temperature as 200C as shown in Fig. 3DLFL.9. and click on  until mesh page.

 

 

Workpiece Object page

 

Observe the imported object mesh (See Fig. 3DLFL.10.). Click on   to 'Material' page.

In object 'Material' selection page ( Fig. 3DLFL.11.), it shows the material name that has been added to the project. Select 'Idea fluid' for the workpiece, then click   to accept it.

Step page

In 'Step' page enter the Number of steps as 200 and ‘Time’ of step increment as 0.04 sec (see Fig. 3DLFL.15.).

 

 

Step page

 

Click on Expert mode , Under Solver setting - Thermal tab select MUMPS solver as shown in Fig. 3DLFL.16. Click on  to Generate DB page.

Generate Database

Click on  , database has been successfully generated.

Run Simulation

Once the database has been generated, switch to the Simulation mode by clicking on   button above the operation tree. Click on the  action label to open the Run Options dialog as shown in Fig. 3DLFL.17.  Use the default Continue Run option to select “Continue from the last step” (from step -1) option and then select the Simulation mode as Interactive and click on  button to run the simulation.
