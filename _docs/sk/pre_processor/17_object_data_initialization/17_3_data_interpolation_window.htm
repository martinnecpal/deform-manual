17.3. Data interpolation window 

[2D, 3D]: Using Data interpolation option user can interpolate object data from another object using different database. (See Fig. 17.3.1.)

Once the database is selected user can select the Step number (see Fig. 17.3.2.), and select the object from which data needs to be interpolated (See Fig. 17.3.3.). Then click on   to interpolate the data. If user wants to interpolate boundary condition data then check "Interpolate boundary conditions" check box.
