10.1.3.4. Lankford coefficient (R value)

This allows the user to specify anisotropic settings using the R-value format of the Lankford coefficient as shown in Fig. 10.1.3.4.1. Lankford coefficient (R value) is useful for evaluating metal flow along an axial direction with small reduction of thickness.
