# 10.1.3.2. Hill’s quadratic (FGHLMN)

This allows the user to specify anisotropic settings using the FGHLMN format of the Hill’s quadratic method as shown in Fig. 10.1.3.2.1. Hill's quadratic yield function overestimates the flow stress of the material under biaxial tension.
