# 10.1.1.13. User defined Tabular data (log)

User can define the flowstress function as any combination of the available Input variables temperature, Stress, Strain, Time, Strain rate, Precipitate size, Precipitate shape and Precipitate Vol. fraction. User can see the graph by selecting graph view tab. Flowstress values are log interpolated in function variables space.
