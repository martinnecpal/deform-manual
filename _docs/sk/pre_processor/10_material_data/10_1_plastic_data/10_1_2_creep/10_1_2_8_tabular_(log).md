# 10.1.2.8. Tabular (Log)

User can define the Creep function as any combination of the available Input variables temperature, Strain, Stress, Time, Strain rate, Grain size, Precipitate size, Precipitate shape and Precipitate Vol. fraction. User can see the graph by selecting graph view tab. Creep values are log interpolated in function variables space. (See tabulate data [Fig. 10.1.2.7.1.](10_1_2_7_tabular_\(linear\).htm#Fig._10.1.2.7.1._Tabulate_data_creep_model_function_definition_window) for definition and graph window)
