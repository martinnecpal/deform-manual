# 16.9. RSE

**[3D]** : The RSE algorithm will automatically be activated during runtime to cut the amount of computation time required to solve the problem. User has to enter the control parameters under RSE window (See Fig. 16.9.1.). The RSE settings work very well for typical forging problems.

In the integrated system there is no room to define these and for rigid plastic models this RSE solver is automatically picked up using default values of these control parameters (non default controls including RSE windows can only be defined using the DEF_RSE.DAT file). Now user can define these in the GUI.
