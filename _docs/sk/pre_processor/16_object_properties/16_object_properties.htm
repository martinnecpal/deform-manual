16. Object Properties

Miscellaneous object parameters, which affect either thermo-mechanical behavior of the object or numerical solution behavior, are specified in the Object-Properties window. (See Fig. 16.1. and Fig. 16.2.)

 

 

2D Object Properties window

 

 

3D Object Properties window

 

Object Properties options are categorized as: 

  • Deformation
  • Thermal
  • Reference
  • Fracture
  • Hardness
  • Heating
  • Symmetry
  • Body Force
  • RSE
  • User

 

For more information, please refer sections 16.1. Deformation properties, 16.2. Thermal properties, 16.3. Reference,  16.4. Fracture Properties, 16.5. Hardness properties, 16.6. Heating Properties,  16.7. Symmetry properties , 16.8. Body Force , 16.9. RSE and 16.10. User.
