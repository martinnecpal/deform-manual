# 16\. Object Properties

Miscellaneous object parameters, which affect either thermo-mechanical behavior of the object or numerical solution behavior, are specified in the Object-Properties window. (See Fig. 16.1. and Fig. 16.2.)

![](../../../assets/Images/Pre-Processor/16_Object_Properties/16_Image001.jpg)

2D Object Properties window

![](../../../assets/Images/Pre-Processor/16_Object_Properties/16_Image002.jpg)

3D Object Properties window

Object Properties options are categorized as: 

  * [Deformation](/docs/sk/pre_processor/16_object_properties/16_1_deformation_properties/)
  * [Thermal](/docs/sk/pre_processor/16_object_properties/16_2_thermal_properties/)
  * [Reference](/docs/sk/pre_processor/16_object_properties/16_3_Reference/)
  * [Fracture](/docs/sk/pre_processor/16_object_properties/16_4_Fracture_properties/)
  * [Hardness](/docs/sk/pre_processor/16_object_properties/16_5_hardness_properties/)
  * [Heating](/docs/sk/pre_processor/16_object_properties/16_6_heating_properties/)
  * [Symmetry](/docs/sk/pre_processor/16_object_properties/16_7_symmetry_properties/)
  * [Body Force](/docs/sk/pre_processor/16_object_properties/16_8_body_force/)
  * [RSE](/docs/sk/pre_processor/16_object_properties/16_9_rse/)
  * [User](/docs/sk/pre_processor/16_object_properties/16_10_user/)
