12.4. 3D Geometry data Editing (GEO TOOL)

3D Geometry tool (GEO TOOL) is used to edit the existing 3D geometry. When we import 3D geometry or when we create geometry using primitive option,  then  link is enabled in 3D Geometry page. When we click on   link (see Fig. 12.4.1.), 3D Geometry Tool will be opened in new window as shown in Fig. 12.4.2. For information related to options of 3D Geometry Tool, please refer 50.1. 3D Geo Tool. After completion of the editing the 3D geometry user can close the 3D Geometry Tool window and the edited geometry is automatically updated in Pre-processor GUI if user saves the geometry while closing the window. 


Fig. 12.4.3. shows the pop-up message box while closing the 3D Geometry tool window, click on to save the modified geometry or click  button to retain original geometry. After saving changes and closing the 3D Geometry tool window, an object selection popup appears as shown in below Fig. 12.4.4. to select the object having edited geometry. User can select the correct object and then click on  to update geometry with modified geometry.
