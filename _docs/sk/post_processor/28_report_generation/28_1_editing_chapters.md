# 28.1. Editing Chapters

Editing chapter and sections options are help full in report generation, as the user needs to change the specific section settings of the report then directly by typing the section number in current section field and pressing the enter key will open that particular section in display window and in Settings modifications window. User defined sections can be added and deleted by using the ![]({{ '/assets/icons/post_icons/mo_add_section_button.jpg' | relative_url }}) (Add new section) and ![]({{ '/assets/icons/post_icons/mo_delete_section_button.jpg' | relative_url }}) (Delete section) icons and also under Section Window .(See Fig. 28.1.1.)

![]({{ '/assets/images/post_processor/28_report_generation/28_1_editing_chapters/image001.jpg' | relative_url }})

Edit sections tool bar

Added new sections using ![]({{ '/assets/icons/post_icons/mo_add_section_button.jpg' | relative_url }}) (Add new section) will add the user defined sections, this can be seen in object tree report tab under the default chapter as shown in Fig. 28.1.2.

![]({{ '/assets/images/post_processor/28_report_generation/28_1_editing_chapters/image002.jpg' | relative_url }})

Object tree reports tab and Chapter settings and modification window with 4 user defined sections

Selecting the other sections from chapter window like summary, graph, state variable, etc will update in edit sections tool bar as shown in Fig. 28.1.3. and these can be accessed using edit sections tool bar for editing.

![]({{ '/assets/images/post_processor/28_report_generation/28_1_editing_chapters/image003.jpg' | relative_url }})

Chapter settings and modification window and Object tree report tab with user defined and other sections

For more information on report sections settings refer chapter [28\. Report Generation](/docs/sk/post_processor/28_report_generation/28_report_generation/) section [28.2. Sections.](28_report_generation.htm#28_2_Sections)
