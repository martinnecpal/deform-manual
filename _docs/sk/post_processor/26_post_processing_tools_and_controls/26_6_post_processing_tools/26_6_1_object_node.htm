26.6.1. Object Node 

[2D, 3D]: The nodes window will display information of the nodes of the currently selected object. This information includes Position (co-ordinates), State variables and Boundary conditions. A different node can be selected through the node double spin box or by selecting a node graphically in the display window. The information will change as the step number displayed changes in the display window (See Fig. 26.6.1.1. and   Fig. 26.6.1.2.). From DEFORM V12, the nodal data can be displayed based on the selected coordinate system selected in State variable page, after changing coordinate system in State variable page, toggle button will be activated and can be used to toggle button between Cartesian coordinate system value and selected new coordinate system value as shown in Fig. 26.6.1.3.
