26.6.11. Coupon Data Extraction 

Qualification of parts requires cut-up evaluation of critical locations for microstructure and mechanical property response. Industry needs to identify critical areas in the part based on forming and heat treatment modeling results. For those critical areas identified to extract modeling state variables a capability has been developed in DEFORM to facilitate this process called Coupon data extraction or Coupons.

For coupon data extraction user need to follow the below mentioned steps,

  1. Select the object for which coupon to be added from the object tree.

  2. Click on the   (Add) button to add the coupons. (See Fig. 26.6.11.1.)

  3. Select the each coupon and locate critical areas in the object by using geometry shape and parameters. (See Fig. 26.6.11.1.)
