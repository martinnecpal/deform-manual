# INICTC

|  (Action Keyword)  
---|---  
|  Last updated on : 07-08-2013  
  
* * *

INICTC Object

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Object |  Object number to initialize |  None  
  
DEFINITION  
---  
INICTC initializes the contact conditions for the specified object.  
  
REMARKS  
---  
INICTC is an action keyword. This contact conditions are initialized when the keyword is read in to the Pre-Processor.
