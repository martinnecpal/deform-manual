PHASLH

(Inter Material data)

 

Last updated on : 08-08-2013

 

PHASLH Mat1, Mat2, IType or Value, Num

Temp(1) PHASLH(1)

Temp(Ndata) PHASLH(Ndata)

  

OPERAND

DESCRIPTION

DEFAULT

Mat1

Material number 1

None

Mat 2

Material number 2

None

IType

Function type

=0 Constant

=1 Function of temperature

=2 Function of atom content

None

Num

Number of data pairs

 

 

DEFINITION

PHASLH is the latent heat that is produced when material 1 transforms into material 2.

 

SYSTEM UNITS: N/mm2

 

EXAMPLE

PHASLH 1 1 1 3

600 100

700 140

800 160
