DEFINITION

PHEQVF specifies the equilibrium volume fraction for precipitation.

 

REMARKS

This keyword sets a flag to indicate how to use the equilibrium volume fraction (i.e. proportional calculation on/off) in FEM calculation.

 

Applicable object types: Plastic, Elastoplastic

Applicable simulation type: Microstructure, Heat treatment, Transformation
