FuncType

Type of function data

= 4: Function of Rodrigues space and temperature (new in v11)

None

 

DEFINITION

ANISO specifies the various Hill’s anisotropic yield function.

 

REMARKS

When Hill’s quadratic (Polycrystal Plasticity) anisotropy yield function is selected (YieldType=4), texture information (crystal type, texture type) should be defined in Material dialog. At each material point of an object, orientation distribution functions will be evaluated as explained in keyword TXTODF.

 

Applicable object types: Plastic, Elastoplastic,

Note: Anisotropy is not available in “Torsion” mode in 2D.
