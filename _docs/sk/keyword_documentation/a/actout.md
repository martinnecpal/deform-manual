# ACTOUT

|  (Action keyword)  
---|---  
_Update History:_ (New definition has been introduced in v10.0) |  Last updated on : 23-07-2013  
  
* * *

ACTOUT Type, FileName

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Type |  Output file type |  10  
FileName |  File name of converted 3D model |  M23_OUTPUT.KEY  
  
DEFINITION  
---  
ACTOUT specifies the type and name of the file that stores the converted 3D model after 2D to 3D object data conversion is complete.  
  
REMARKS  
---  
This is action keyword usually written in DEF_MULTI.INI which is input file to M23.EXE binary which converts 2D model to 3D. Typically, this file name is assigned as M23_OUTPUT.KEY.
