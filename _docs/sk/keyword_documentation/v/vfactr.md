# VFACTR

|  (Simulation control – 2D)  
---|---  
|  Last updated on : 08-08-2013  
  
* * *

VFACTR SwitchOn

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
SwitchOn  |  =**0** : Off; =**1** : On  |  0  
  
DEFINITION  
---  
VFACTR determines whether inter object heat transfer by radiation is computed in the simulation.
