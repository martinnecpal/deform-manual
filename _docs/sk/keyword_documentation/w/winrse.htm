DEFINITION

WINRSE specifies the rigid window which defines a specific region where plastic deformation is not expected. This rigid window is optional input for RSE solver. Multiple rigid windows can be defined per object.

 

REMARKS

If only some portion of deforming object undergoes plastic deformation often found in Cogging simulation, using the rigid window can help RSE to improve the computational efficiency by reducing the effort in searching a plastically deforming zone.

 

Applicable object types: Plastic
