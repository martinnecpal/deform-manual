# WRTOBJ

|  (Action keyword)  
---|---  
|  Last updated on : 08-08-2013  
  
* * *

WRTOBJ Obj, NewObj,

File_name

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Obj |  Object to be written |  None  
NewObj |  New object number it is written to |  None  
File_name |  Name of file to be written to |  None  
  
DEFINITION  
---  
WRTOBJ writes out object information for a specified object. Bye specifying NewObj, as a different number. The object number can be changed to a new number.
