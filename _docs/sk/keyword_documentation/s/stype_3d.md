OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Stype |  Simulation Type = **1** Largrangian Incremental = **2** Largrangian Discrete (obsolete) = **3** Steady-state Machining = **4** Steady-state Extrusion = **5** ALE Rolling = **6** ALE Ring-Rolling = **7** ALE Extrusion = 8 Electro- Magnetic forming = 9 ALE Stir Welding = 10 ALE Spinning = 11 ALE Spinning(express) = 12 ALE Tube Piercing |  1  
  
DEFINITION  
---  
STYPE specifies the simulation type which is to be used.  
  
REMARKS  
---  
Applicable simulation types: Isothermal Deformation, Heat Transfer, Non-Isothermal Deformation
