DEFINITION

STRAIN defines the effective strain of each element of an object.

 

REMARKS

The keyword format varies depending on strain output selection made at ELMNOD.

For element output (ELMNOD = 0) option, effective strain at the centroid of each element is written.

For dual output (ELMNOD = 2) option, effective strain at the centroid of element and at node are written.

For brick element case with integration point (ELMNOD = 3) option, effective strain values at eight integration points in element are written.

 

Note: This integration point option is valid only for brick element whose strain values are evaluated at 8 integration points per element during FEM calculation. For Tetrahedral element, integration point output option is same as elemental output option.   

 

Applicable object types:Plastic, Elastoplastic, Porous
