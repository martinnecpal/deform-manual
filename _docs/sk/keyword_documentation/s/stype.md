# STYPE

|  (Simulation control - 2D)  
---|---  
|  Last updated on : 17-08-2017  
  
* * *

STYPE SType

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Stype |  Simulation Type = **1** Largrangian Incremental = **2** Largrangian Discrete (obsolete) = **3** Steady-state Machining |  1  
  
DEFINITION  
---  
STYPE specifies the simulation type which is to be used.  
  
REMARKS  
---  
Currently the incremental simulation type is the only simulation type available. Applicable simulation types: Isothermal Deformation, Heat Transfer, Non-Isothermal Deformation
