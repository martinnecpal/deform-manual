REFPOS

(Simulation Control - 2D)

 

Last updated on : 09-08-2013

 

REFPOS Object, Method, Var1, Var2

  

OPERAND

DESCRIPTION

DEFAULT

Object

Object number

None

Method

Distance tolerance for critical nodes

(0 = None, 1 = coordinate , 2 = node)

None

Var1

X coordinate or Node number

None

Var2

Y coordinate or Blank

None

 

DEFINITION

REFPOS sets reference points for an object.

 

REMARKS

REFPOS is to be used in conjunction with MDSOBJ which sets the distance between two objects.
