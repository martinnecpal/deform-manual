# CNTRA2

|  (Object data – 3D)  
---|---  
_Update History:_ (New definition has been introduced in v10.0) |  Last updated on : 12-08-2013  
  
* * *

CNTRA2 Obj, arg1...arg7

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Obj |  Object number |  None  
Arg 1-3 |  Rotational axis point |  None  
Arg 4-6 |  Rotational axis vector |  None  
Arg 7 |  Current angle |  None  
  
DEFINITION  
---  
CNTRA2 specifies the center of rotation for objects which move with a rotational velocity.  
  
REMARKS  
---
