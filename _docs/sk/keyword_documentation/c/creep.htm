DEFINITION

CREEP specifies the creep model being considered in a simulation.

 

REMARKS

Creep is defined as the time-dependent permanent deformation under stress that usually occurs at high temperatures. It is common in applications where the material undergoes cyclic loading. The keyword CREEP must only be used with elasto-plastic objects. In addition, the keyword CREEP must be activated for each object.

 System Unit: (s-1)

 

Applicable simulation modules: Deformation, Thermal

 

Applicable object types: Elastoplastic
