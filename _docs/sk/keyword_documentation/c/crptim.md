# CRPTIM

|  (Object Data)  
---|---  
|  Last updated on : 23-07-2013  
  
* * *

CRPTIM Object Creep

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Object |  Object number |  None  
Creep |  Creep calculation =**0** Not active = **1** Active |  0  
  
DEFINITION  
---  
CRPTIM specifies the activation of creep calculation in elasto-plastic object.  
  
REMARKS  
---  
When creep calculation is turned on for a given object, the selected creep model in the material dialog will be used to calculate creep behavior. The creep calculation is only applicable to elasto-plastic objet. Applicable simulation types: Heat treatment
