# CNTERR

|  (Simulation Control - 2D)  
---|---  
|  Last updated on : 29-07-2013  
  
* * *

CNTERR ReleaseMethod, Distance

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
ReleaseMethod |  Contact release method (=**0** : default; =**1** : based on distance) |  0  
Distance |  Contact release distance |  0  
  
DEFINITION  
---  
CNTERR specifies the contact release policy.
