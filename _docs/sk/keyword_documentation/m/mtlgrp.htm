DEFINITION

MTLGRP specifies the material number associated with each element.

 

REMARKS

Each material number represents a set of material properties used to simulate the material behavior.

An object may be comprised of elements having different material properties.

Applicable Object types: Rigid, Elastic, Plastic, Elastoplastic, and Porous.
