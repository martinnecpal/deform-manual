DEFINITION

MORP specifies the source, target, and output geometries for morphing with associated DOE variable numbers.

 

REMARKS

This is action keyword which will create DEF_MORP2.INI and spawn DEF_MPRP2.EXE. The newly created geometry as a result of morphing will be loaded by action keyword GFREAD. If nominal/source object has mesh, new mesh will be created from action keyword DEFAMG.

 

DOE variable numbers will be converted to weighting factors by parsing.
