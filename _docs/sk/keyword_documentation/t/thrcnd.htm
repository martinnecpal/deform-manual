DEFINITION

THRCND specifies the thermal conductivity of a particular material.

 

REMARKS

The thermal conductivity may be specified as a constant value as a set of discrete temperature/conductivity data pairs.

 

If Ftype = 0 use the operand Conductivity.

 

If Ftype = 1 use the operands Ndata, Temp(i), Conductivity(i). Each temperature/thermal conductivity pair should be provided on a separate line. When temperatures lie within the specified data range, linear interpolation is used to determine the corresponding thermal conductivity. When temperatures lie outside the specified data range, linear extrapolation is used to determine the corresponding thermal conductivity.

 

If Ftype = 2 use the operands N1 and Density (i).

 

If Ftype = 3 use the operands N1 and Atom(i). It should be noted that the atom variables means the weight percent of solute in the material.

 

If Ftype = 4 use the operands N1, N2, Temp (i), Atom (i).
