# TMPLMT

|  (Object data)  
---|---  
|  Last updated on : 08-08-2013  
  
* * *

TMPLMT Obj# TmpLmt 

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Obj# |  Object number |  None  
TmpLmt |  Temperature limit |  None  
  
DEFINITION  
---  
TMPLMT, also referred to as the truncation temperature, specifies a temperature upper limit of an object. This is often used to prevent temperature from rising above the range of available material data.  
  
REMARKS  
---  
Applicable Simulation Modes: Deformation Applicable object types: [Plastic](../../pre_processor/11_general_object_data_definition/11_general_object_data_definition.htm#11.4.1_Plastic), [Elastoplastic](../../pre_processor/11_general_object_data_definition/11_general_object_data_definition.htm#11.4.3._Elasto-plastic) and [Porous](../../pre_processor/11_general_object_data_definition/11_general_object_data_definition.htm#11.4.4._Porous)
