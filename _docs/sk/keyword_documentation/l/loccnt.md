# LOCCNT

|  (Inter object data – 3D)  
---|---  
_Update History:_ V11.1 - LOCCNT has been introduced only for 3D |  Last updated on : 21-05-2016  
  
* * *

LOCCNT Object1, Object2, Flag

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Object1 |  Object number of first object |  None  
Object2 |  Object number of second object |  None  
Flag |  Contact option =**0** do NOT use local contact algorithm =**1** use local contact algorithm |  None  
  
DEFINITION  
---  
LOCCNT specifies the contact algorithm between two objects.  
  
REMARKS  
---  
Contact search in FEM will only look for the nodes inside friction window
