# FRAE2H

|  (Inter Material data)  
---|---  
|  Last updated on : 05-08-2013  
  
* * *

FRAE2H Material, Factor

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Material |  Material number |  None  
Factor |  Fraction of mechanical energy converted to heat energy |  0.9  
  
DEFINITION  
---  
FRAE2H specifies the fraction of mechanical work converted to heat.  
  
REMARKS  
---  
The conversion fraction would typically be 0.9 ≤ Factor ≤ 0.95. Applicable simulation types: Non-Isothermal Deformation.
