# KFWRIT

|  (Action keyword)  
---|---  
|  Last updated on : 07-08-2013  
  
* * *

KFWRIT

filename

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
filename |  Filename of keyword file to write |  None  
  
DEFINITION  
---  
KFWRIT writes the data in preprocessor into a keyword file.  
  
REMARKS  
---  
It is an action keyword placed in either an automatic script file or in a Master file.
