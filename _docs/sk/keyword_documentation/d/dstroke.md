# DSTROKE

|  (Action keyword)  
---|---  
_Update History:_ (New keyword in v11) |  Last updated on : 23-07-2013  
  
* * *

DSTROKE Object, DeltaStroke

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Object |  Object Number |  None  
DeltaStroke |  Object stroke in X |  0  
  
DEFINITION  
---  
DSTROKE specifies the change in remaining stroke of objects with a movement control boundary constraint.  
  
REMARKS  
---  
DSTROKE is an action keyword. It can be used to modify current stroke or remaining stroke by a given value along the object's movement direction.
