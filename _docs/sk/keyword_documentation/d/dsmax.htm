DSMAX

(Simulation control)

 

Last updated on : 29-07-2013

 

MaxStrokeStep

  

OPERAND

DESCRIPTION

DEFAULT

MaxStrokeStep

Maximum stroke of the primary object per time step

0

 

DEFINITION

DSMAX limits the displacement of the primary object in one time step. A new time step will be initiated when the displacement of the primary object reaches MaxStrokeStep.

 

REMARKS

A non-zero value must be assigned to either DSMAX or DTMAX, but values cannot be assigned to both.

 

DSMAX and DTMAX are two of several parameters used to control the size of time steps. Other keywords which effect time step size include: DEMAX, DVMAX, and SLDERR. A new time step will be generated when the criteria specified in any of these keywords has been satisfied.

 

Typically, the value of MaxStrokeStep should correspond to an average strain increment of about 0.01, i.e. a one percent height reduction.

 

If MaxStrokeStep = 0, DSMAX will not be used as a step size parameter.

 

The time steps generated by DSMAX will be recorded in the DEFORM database.

 

Applicable simulation types: Isothermal Deformation, Non-Isothermal Deformation
