# DATOM

|  (Action keyword)  
---|---  
|  Last updated on : 29-07-2013  
  
* * *

DATOM Iobj, Numnp

1 c1

n cn

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
Iobj |  Object Number |  None  
Numnp |  Number of nodes |  None  
  
DEFINITION  
---  
DATOM specifies atom content at a node. SYSTEM UNIT: (wt%)  
  
REMARKS  
---  
The initial carbon content should be specified at each node if a carburizing process is simulated. In addition, the atom content should be specified initially if the user uses material and inter-material data that depend on atom content. Applicable Simulation Module: Microstructure Applicable Simulation Modes: Diffusion Applicable Object Types: ALL except Rigid.
