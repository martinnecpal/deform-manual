# DBREAD

|  (Action keyword)  
---|---  
|  Last updated on : 29-07-2013  
  
* * *

DBREAD step#

* * *

OPERAND |  DESCRIPTION |  DEFAULT  
---|---|---  
step# |  = **0** Read the last step Otherwise, read specific step number |   
Filename |  Specific file name |   
  
DEFINITION  
---  
DBREAD allows the user to input an arbitrary database step into the preprocessor. It is used in a Keyword file.  
  
REMARKS  
---  
It is an action keyword placed in Keyword files that drives the preprocessor to read a database file.
