DEFINITION

OBJPOS positions objects specified by the variables in the keyword.

 

REMARKS

OBJPOS is an action keyword that positions objects when it is read in to the Pre-Processor. This keywords main use is for multiple operations.

 

For flipping, the object is flipped with respect to a line crossing point (X1, Y1) and (X2, Y2).

If N is not zero, OBJPOS will perform coupled positioning and read second line which is the list of object numbers. (supported from 2DV9.1 beta)
