# 3.2. Monitorovanie licencií

**Ako skontrolovať spustené úlohy a dostupné licencie zo servera**

Po spustení služieb a spustení programu DEFORM je možné pomocou monitorovania licencií programu DEFORM získať prístup k stavu zaškrtnutých licencií podľa programov a k dostupným licenciám, ako je uvedené nižšie v ponuke Štart programov. (Pozri obr. 3.2.1. a obr. 3.2.2.)
