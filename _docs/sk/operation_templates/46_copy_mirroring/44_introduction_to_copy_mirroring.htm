46 Introduction to Copy/Mirroring

Copy/Mirroring template can be used to mirror/copy an object which is having a planar symmetry along with state variable data, currently the template does not support rotational symmetry. Several times we come across processes those exhibit symmetry in their setup up to certain stages and later it may not have symmetry in its geometry depending upon the final product requirement. User need not model whole object but can model the process as symmetry for those stages where symmetry exists and use copy/ mirroring operation to copy/ mirror the object into whole object or delete the mirrored/copied object and use it in further stages.
