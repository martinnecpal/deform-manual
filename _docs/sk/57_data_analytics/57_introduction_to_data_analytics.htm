57. Introduction to DEFORM Data Analytics

DEFORM Data Analytics provides a way for users to analyze, compare and model data. Data can be from experiments or FEM output from DEFORM simulation databases. Data can be provided in simple .txt file and .csv file.


DEFORM Data Analytics can also be used to perform sensitivity analysis of the input parameters of the process or compare FEM results with experimental data. 
Analysis of the data is performed by visualizing the data using plots (such as histogram and scatter plots) and computing the statistics.


Data modeling is essentially curve fitting. Several options such as linear, cubic, quadratic, gaussian and neural network methods are provided to model the data. There is also an option to fit data to a user defined equation.


DEFORM Data analytics models can extend the FEM by including the data models from DEFORM Data Analytics in post-processing. DEFORM Data analytics models can also be used as a surrogate models when FEM simulation is computationally expensive.
